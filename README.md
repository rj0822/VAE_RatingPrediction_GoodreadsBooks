The implementation is developed based on the book reviews within poetry category from the Goodreads books dataset.

The dataset is available at https://cseweb.ucsd.edu/~jmcauley/datasets/goodreads.html (goodreads_reviews_poetry.json.gz) or https://drive.google.com/drive/folders/1fIKpy080ClTv7pdimVv6kTSZqZE437iI?usp=drive_link.
